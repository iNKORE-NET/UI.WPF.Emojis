Survey Tool instances:

instructions

Shows the various surveytool instances you are configured to watch. The Details button will show a snapshot of the Admin Panel. Note that details are normally paused (to save server load!) and can be manually refreshed with the Refresh Details button. If you know you want to capture each detail, uncheck the Pause Details button.

For example, unchecking Pause Details would allow inspection of the last known stack traces at the point the server stopped responding.

These tools will let you update the SurveyTool.
